계정 및 데이터 삭제

장기대전 앱에서 계정 및 관련 데이터를 삭제하는 방법을 안내합니다.

방법 1 — 앱 내에서 직접 삭제

앱에 로그인한 상태에서 아래 경로로 이동하면 즉시 계정을 삭제할 수 있습니다.

설정 → 계정 → 계정 삭제

삭제 즉시 모든 데이터가 서버에서 제거됩니다.

방법 2 — 이메일로 요청

앱에 접근할 수 없는 경우 이메일로 삭제를 요청하실 수 있습니다.
요청 접수 후 영업일 기준 3일 이내 처리됩니다.

이메일 요청 시 포함할 내용
  • 가입 시 사용한 Google 계정 이메일
  • 앱 내 닉네임
  • 삭제 요청 사유 (선택)
삭제 요청 이메일 보내기

삭제되는 데이터

  • 닉네임, 프로필 사진 URL
  • 게임 기록 (승/패/무, 레이팅)
  • 토큰 내역
  • Firebase 인증 계정

※ 삭제된 데이터는 복구할 수 없습니다.
